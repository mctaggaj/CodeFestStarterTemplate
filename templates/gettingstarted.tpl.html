Getting Started: Common tasks

  1. To add a new page, under templates add a new html file named page.tpl.html (replace "page" with the name of your page)

  2. In the app.js file under js add a new route to the $routeProvider function

    config(['$routeProvider', function($routeProvider) {
        $routeProvider.when('/home', { templateUrl: 'templates/home.tpl.html' });
        $routeProvider.when('/bootstrap', { templateUrl: 'templates/bootstrap.tpl.html' });
        $routeProvider.when('/angular', { templateUrl: 'templates/angular.tpl.html' });
        $routeProvider.when('/template', { templateUrl: 'templates/template.tpl.html'});
        $routeProvider.when('/testing', { templateUrl: 'templates/testing.tpl.html'});
        $routeProvider.when('/page', { templateUrl: 'templates/page.tpl.html' });
        $routeProvider.otherwise({ redirectTo: '/home' }); }]);

    In the index.html file add a new list item to the navbar

    <ul class="nav navbar-nav">
        <li ng-class="{active: selectedPage==1}" ng-click="selectedPage=1">
            <a href="#/bootstrap">bootstrap</a>
        </li>
        <li ng-class="{active: selectedPage==2}" ng-click="selectedPage=2">
            <a href="#/angular">angular</a>
        </li>
        <li ng-class="{active: selectedPage==3}" ng-click="selectedPage=3">
            <a href="#/gettingstarted">getting started</a>
        </li>
        <li ng-class="{active: selectedPage==3}" ng-click="selectedPage=4">
            <a href="#/testing">testing</a>
        </li>
        <li ng-class="{active: selectedPage==5}" ng-click="selectedPage=5">
            <a href="#/page">page</a>
        </li>
     </ul>

    Customize the new page to add your new functionality.

  3. You can remove any pages that don't make sense for your app (e.g. the bootstrap and angular example pages).
  4. To install third party controls, you can use either the Node Package Manager (NPM) or the Bower Package Manager. Which one you choose to use depends on the library you are trying to add. To do this, open a Command Prompt and run one of the following example commands (you have to modify the name of the dependency you want to install):
    • bower install angular-google-maps
    • npm install underscore
