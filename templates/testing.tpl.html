Unit Testing

For unit testing we use jasmine tests run by karma.
Unit tests are located in the test\unit folder and can be started by running the test_unit.bat file

Integration (End-to-End) Testing

For integration testing we use protractor.
Integration tests are located in the test\e2e folder and can be started by running the test_e2e.bat file (IMPORTANT: The Visual Studio project must be running for protractor to work)

For more info, read the getting started docs.
